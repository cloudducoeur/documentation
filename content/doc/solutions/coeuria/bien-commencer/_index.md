Vous souhaitez utiliser CoeurIA, suivez cette documentation pour bien commencer.

{{% steps %}}

### Obtenir un accès

Pour commencer, vous devez avoir un compte `@restosducoeur.org` pour accéder à CoeurIA. Le SSO est actif sur CoeurIA.

{{< callout type="info" >}}
  Si vous rencontrez un souci pour vous connecter, veuillez lire [cette documentation](/doc/commun/aide/).
{{< /callout >}}

### Se connecter à l'interface

Une fois que vous avez obtenu vos accès, vous allez pouvoir accéder à l'interface de CoeurIA et vous authentifier via SSO.
